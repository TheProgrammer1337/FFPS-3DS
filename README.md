# FFPS-3DS
## Overview
This project is a fan-made remake of Freddy Fazbear's Pizzeria Simulator for Nintendo 3DS consoles.

## Features
- Intro 8-bit minigame
- Blueprint mode (somewhat)
- 7/18 blueprint mode minigames
